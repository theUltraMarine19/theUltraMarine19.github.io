Research Experience

Imaging Techniques with Raman Spectroscopy

Bachelor's Thesis, IIT Bombay
Prof. Ajit Rajwade, Assistant Professor, IIT Bombay

Typical Raman spectrocopy takes a very long acquisition time and is used for diagnosing critical diseases like cancer. The aim of this project is to reduce the acquisition time without compromising on quality. We are trying to learn a compact representation of paraffin subspace for spectral separation of biopsy sample using Non-negative Sparse Coding, employing Blind Dictionary Learning, coupled with PCA for signal and noise separation. The chief innovation here is performing inpainting to enable compressed sensing of Raman spectral images, which will help speedup image acquisition. This has never been applied for this purpose before. We are currently working towards implementing deep neural networks for inpainting, and super-resolution for lower resolution Raman images with specific sampling heuristics

Benchmarking of Software Switches

Research Project, IIT Bombay
Prof. Mythili Vutukuru, Assistant Professor, IIT Bombay

VPP and Open vSwitch are currently the fastest DPDK -based software switches out there. The aim was to determine the minimal resources required for optimal performance of a switch for different use cases. I conducted experiments independently for testing latency, throughput, efficiency in terms of cycles per packet with increasing cores, routing table entries and hierarchical cache sizes. These were tested using uniform and skewed(Gaussian) traffic loads of 10 Gbps. I analyzed VPP’s batch packet processing paradigm and tested batch size as a function of different parameters. This also involved exploring Cisco Express Forwarding implemented using multiway prefix trees in VPP, which has been patented by Cisco

Characteristics-Tailored Summary Generation

Adobe Research, Research Intern
Dr. Balaji Vasan Srinivasan, Sr Reseach Scientist, Adobe Research

Unlike typical abstractive text summarisation, the aim was to tune our summaries to characteristics like being more formal as required by news agencies or focus on financial aspects as desired by corporate organisations. We used Facebook AI Research’s Convolutional seq2seq model for translation adapted to the purpose of characteristic-driven text generation. We experimented at different stages of this framework, like modifying attention layers to focus on specific input embeddings for topic-tuned summaries, altering beam search paradigm for tweaking decoder state probability distributions, which would help enhance word-level features like descriptiveness and also help incorporate specific vocabulary words. We also tested the token-based learning approach for length based summarization with different bins of short, medium and long. We also incorporated a Reinforcement Learning term in the loss function which involved the readability score metric and achieved a significant increase in ROUGE scores. We implemented the above insights on the pointer-generator framework and submitted the same to AAAI 2019. I received a Pre-Placement Offer from Adobe Systems for the same. We have resubmitted the work after some minor improvements to NAACL 2019, hence the report is not made available in the public domain.

Optimizing Performance of Model-counting Algorithms

Research Project, NUS
Prof. Kuldeep Meel, Assistant Professor, NUS

The task involved a study of different model-counting algorithms, which enumerate solutions to a boolean formula. The aim was to identify performance bottlenecks in the implemented model for optimization. I made a critical study of SPARSE-COUNT algorithm and extended the same using GMP, MPFR libraries to support arbitrarily large number of variables, where computations overflow even the largest data type for integers in C and multi-precision computations. I implemented the above in the ApproxMC framework which is another similar framework for model-counting, using θ(logn) low-density parity constraints with tolerance guarantees for results within a specified confidence interval. The results were validated using IJCAI’16-CMV benchmarks.

Domain-specific Customer Care Chatbot

Research Intern, Philips Innovation Campus
Dr. Rajendra Singh Sisodia, Principal Scientist, Philips Innovation Campus

Modern chatbots perform well in conversations comprising simple question-answer pairs. The aim was to develop a semantic control algorithm to track context switches to predict favourable next steps in the conversation. We designed a chatbot leveraging word2vec, Latent Semantic Indexing, Latent Dirichlet Allocation to return a list of confidence-weighted topics relevant to user query, coupled with tf-idf weights and word n-grams for improving accuracy. We tried to incorporate a probabilistic finite automata to model conversation state changes, guided by sentiment scores. For this purpose, we built a linear SVM based emotion classifier. To optimize for faster responses, I leveraged building ontologies from XML, RDF sources using Stanford's Protege together with SPARQL queries for fetching fast respsonses.
